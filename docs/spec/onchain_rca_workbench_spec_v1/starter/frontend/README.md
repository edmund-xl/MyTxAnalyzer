# Frontend Starter

Codex should initialize a Next.js 14 TypeScript app here.

Required pages:

```text
/app
├── page.tsx                         Dashboard
├── cases/new/page.tsx               New Analysis
├── cases/[caseId]/page.tsx          Case Overview
├── cases/[caseId]/timeline/page.tsx Timeline
├── cases/[caseId]/evidence/page.tsx Evidence
├── cases/[caseId]/findings/page.tsx Findings
├── cases/[caseId]/reports/page.tsx  Reports
└── networks/page.tsx                Network Config
```

Use:

- shadcn/ui
- TanStack Query
- React Flow
- Monaco Editor
